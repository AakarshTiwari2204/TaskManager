# 🚀 TaskMaster - To-Do List Web Application

A modern, feature-rich to-do list web application designed to help you organize your daily tasks effectively and boost your productivity.

## ✨ Features

### 🎯 Core Functionality
- **📝 Task Management** - Create, edit, delete, and organize tasks
- **📊 Status Tracking** - Track tasks through To Do, In Progress, and Completed stages
- **📅 Due Date Management** - Set and track deadlines for your tasks
- **🔍 Search & Filter** - Find tasks quickly with powerful filtering options
- **📱 Responsive Design** - Works seamlessly across all devices

### 🎨 User Experience
- **🎨 Modern UI** - Beautiful, intuitive interface with smooth animations
- **🌈 Dark/Light Mode** - Comfortable viewing in any lighting condition
- **⚡ Real-time Updates** - Instant sync across all your devices
- **🔔 Notifications** - Stay on top of your deadlines and tasks

### 🔐 Security & Privacy
- **🔐 User Authentication** - Secure sign-up and login functionality
- **🛡️ Data Protection** - Your data is encrypted and secure
- **👤 Personal Space** - Each user has their own private task list

## 🚀 Technology Stack

### 🎯 Frontend
- **⚡ Next.js 15** - React framework for production
- **📘 TypeScript 5** - Type-safe JavaScript
- **🎨 Tailwind CSS 4** - Utility-first CSS framework
- **🧩 shadcn/ui** - High-quality, accessible components
- **🎯 Lucide React** - Beautiful icon library
- **🌈 Framer Motion** - Smooth animations and interactions

### 🗄️ Backend & Database
- **🗄️ Prisma** - Next-generation ORM
- **🔐 JWT Authentication** - Secure user authentication
- **🍪 Cookie-based Sessions** - Secure session management
- **💾 SQLite** - Lightweight, reliable database

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Set up the database
npm run db:push

# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to see your application running.
